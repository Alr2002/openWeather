# One Call API 3.0 Documentation Test Project
## Introduction
One Call API 3.0 is one of the most advanced APIs, allowing you to build your own\
weather forecast system with high accuracy and display a large number of environmental parameters.
## Purpose
The goal of the project is to demonstrate the skills of a technical writer.
This document provides comprehensive overview of the Call API 3.0, including available endpoints,\
request/response formats, authentication requirements, and other technical specifications intended\
to assist developers and system integrators in utilizing the API effectively. 
## Link to the official One Call API 3.0 documentation
[https://openweathermap.org/api](https://openweathermap.org/api)
