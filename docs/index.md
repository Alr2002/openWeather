# One Call API 3.0 Documentation Test Project

## Introduction

One Call API 3.0 is one of the most advanced APIs, allowing you to build your own
weather forecast system with high accuracy and display a large number of environmental parameters.

## Use cases

**Local weather apps:**  Developing apps for iOS, Android, or the web that display the current temperature, humidity, and a multi-day forecast based on the user’s location.

**Notifications and Widgets:** Developing system notifications in the status bar or desktop widgets  that keep users informed of weather changes.

**Logistics and Transportation:** Using the API to plan routes that account for weather hazards (black ice, heavy rain) along the entire route

## Versioning and updates

One Call API version 3.0

## Purpose

The goal of the project is to demonstrate the skills of a technical writer.
